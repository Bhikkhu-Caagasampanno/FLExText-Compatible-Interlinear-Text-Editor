# FLExText-Compatible-Interlinear-Text-Editor

An interlinear text editor for .flextext files, still a work in progress. supports 3 lines of editing (source, morphological breakdown, gloss) and a section by section free translation. 

This is built as a tool for assisting in  the translation of texts from Pāli to English. I haven't added a feature to build .flextext files from scratch, yet.

It started out as just a script to reformat a Fieldworks interlinear text .xml file to work with the expex LaTeX package for publishing linguistics papers, so that I can eventually get my translations printed, and morphed into this as I got frustrated with Fieldworks Language Explorer's UI clipping issues on linux with large font sizes.

The lexicon editing feature is only partially functional, a rework is in progress.

Export as .flextext, .csv, LaTeX expex formatting.
